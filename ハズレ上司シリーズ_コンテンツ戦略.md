# ハズレ上司シリーズ コンテンツ戦略

## コンセプト

リス丸（@Risumaru153713）が「上司ガチャ」のカード画像を使って、**JTCあるある上司**を1人ずつ紹介するシリーズ投稿。共感度の高い"あるある系"から始めて、徐々にレアリティ（ヤバさ）を上げていき、最終投稿でゲームリリースを告知する。

## シリーズ設計

- **投稿数**: 全8投稿（7キャラ紹介 + 1ゲームリリース）
- **ペース**: 2日に1回（約2週間で完結）
- **統一ハッシュタグ**: `#上司ガチャ` `#ハズレ上司シリーズ`
- **画像**: 各投稿にカード画像を1枚添付
- **投稿時間**: 21:00〜22:00（ゴールデンタイム、社会人の共感が最大化）


### エンゲージメント最大化のポイント

1. **画像は必須** — カード画像がフックになる。タイムラインで目を引く
2. **フレーバーテキストを冒頭に** — セリフで引っかけてから詳細へ
3. **RTしたくなる問いかけ** — 「この上司引いたことある？」的なCTA
4. **レアリティ上昇で期待感** — Post 5からUR突入で「まだ上がいる」感
5. **最終回の予告** — Post 7で「明日、伝説のハズレ上司が降臨」と煽る
6. **リリース投稿は引用RTで拡散** — 過去のシリーズ投稿を引用RTして「全部引けるゲーム作った」

### 追加施策

- シリーズ途中でアンケート投稿：「一番ハズレだと思う上司は？」（4択）
- リリース後：「最初に引いた上司をシェアして」で自然拡散を狙う
- フォロワーの反応が良いキャラは追加ポストで深掘り
